## How style modifies the baseline

### 1) Path shaping (apex + bias)

We create a styled reference inside the lane:

$$
\tilde{\gamma}(s)=\gamma(s)+d(s;p_{\text{apex}},p_{\text{bias}})\mathbf{n}(s)
$$

## Pseudocode (teaching version)

```text
Inputs: state, path gamma(s), curvature kappa(s), grip mu, style params p
1) Build styled path gamma_tilde(s) using apex/bias offsets
2) Pick lookahead target on gamma_tilde(s) at distance L
3) Compute steering using a geometric method (Pure Pursuit or Stanley)
4) Compute curvature-based target speed v_t(s), scaled by risk
5) Track speed with PID to output a_x
6) Apply rate/jerk limits controlled by smoothness
Outputs: delta, a_x
